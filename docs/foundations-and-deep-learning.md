# Foundations and Deep Learning

Extended reading for core machine learning and deep learning. Items in the main [README](../README.md) are not repeated here; start there for the essentials.

Labels:

- Cost: `[Free]`, `[Paid]`
- Level: `[Beginner]`, `[Intermediate]`, `[Advanced]`

## Foundations

- `[Paid] [Intermediate] [Book]` [Elements of Statistical Learning (2009)](https://link.springer.com/book/10.1007/978-0-387-84858-7). More theoretical follow-on once `ISL` starts to feel too lightweight.
- `[Free] [Beginner] [Reference]` [Machine Learning Yearning (2018)](https://www.mlyearning.org/). Useful for framing error analysis, data strategy, and project decisions.
- `[Free] [Advanced] [Reference]` [Probabilistic Machine Learning: Advanced Topics (2023)](https://probml.github.io/pml-book/book2.html). Broad modern reference for probabilistic methods.

## Neural Networks and Representation Learning

- `[Free] [Intermediate] [Book]` [Deep Learning (2016)](https://www.deeplearningbook.org/). Canonical theory-oriented deep learning reference.
- `[Free] [Intermediate] [Book]` [Understanding Deep Learning (2023)](https://udlbook.github.io/udlbook/). Modern treatment of optimization, generalization, and common architectures.
- `[Free] [Beginner] [Book]` [A Brief Introduction to Neural Networks](https://www.dkriesel.com/en/science/neural_networks). Accessible walkthrough of perceptrons and backpropagation.

## Courses

- `[Free] [Intermediate] [Course]` [MIT 6.S191: Introduction to Deep Learning](https://introtodeeplearning.com/). Compact course with strong coverage of core deep learning topics.

## Study Order

1. Start with `Introduction to Statistical Learning` (in README) if you are new to ML.
2. Read `Neural Networks and Deep Learning` (in README) or `Dive into Deep Learning` (in README) for neural model intuition.
3. Use `Machine Learning Yearning` while building projects, not before.
4. Move to `Elements of Statistical Learning`, `Deep Learning`, or `Probabilistic Machine Learning` only after the basics are stable.
